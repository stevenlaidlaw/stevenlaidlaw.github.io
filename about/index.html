About

2016-10-06

I like creating stuff, but I can never settle down on what, so I just try to do all the things. I’d probably get a lot more done if I didn’t spread my focus so much, but what can you do?

Writing

I write novels and screenplays. You can find my currently available novels in the ‘Works’ section of this website.

Code

I’ve built several websites including the one you’re on right now, plus a few other things.

  • LolSheet - A League of Legends web app designed to help you improve at the game, build using Universal React/Express code.
  • Phorg - A photo organisation app built using Node.JS on Electron.
  • SimpleDonut - A stand-alone donut generation library that creates SVGs for simple charts.

The rest of the things I’m publicly working on are available at my Github page.

Film

I’m currently in the process of creating a few short films. Watch this space for more.

Music

Every now and then I go on sprees and make a bunch of music, then stop for a few months again. Go to my Soundcloud page to listen for free, or else some of my stuff is available on Spotify, iTunes, Google Music, Amazon, etc.

Personal

I live in Western Australia with my wife, two daughters, and a pair of trouble making cats.

You can find my latest updates on this site and Twitter.
