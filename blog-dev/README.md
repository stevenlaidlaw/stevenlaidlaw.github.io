# The Blog

Created with Hugo

## Commands

### Create a post:

```sh
hugo new posts/my-first-post.md
```

### The front-matter:

```md
---
title: "My First Post"
date: 2019-03-26T08:47:11+01:00
draft: true
---
```

Drafts don't get published.

### Dev

```sh
hugo server -D
```

### Build

```sh
hugo -D
```
